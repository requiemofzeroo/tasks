## **Задание 1**
Анализатор статус-кодов API
Бизнес-требование: Напишите функцию-помощник для проверки ответов бэкенда.
Что сделать в коде:
1. Создайте функцию `is_server_ok(status_code)`.
2. Если `status_code == 200`, функция должна вернуть булево значение `True`.
3. Во всех остальных случаях функция должна вернуть `False`.
4. В основном коде вызовите функцию, передав туда число 500, сохраните результат в переменную и проверьте её через `if...else`: если вернулся False, выведите `[FAIL] Сервер упал!`.
⚠️ Напоминание: внутри функции используем только `return`, никаких print()!

## **Решение**

def is_server_ok(status_code):

    if status_code == 200:
        return True
    else:
        return False

rezult=(is_server_ok(500))

if rezult == False:

    print("[FAIL] Сервер упал!")
else:

    print("[SUCCESS] Серевер работает корректно")


## **Задание 2**
Генератор уникальных email-адресов
Бизнес-требование: Чтобы регистрировать новых пользователей в автотестах, нам нужен генератор случайных почтовых ящиков.
Что сделать в коде:
1. Напишите функцию `generate_qa_email(prefix)`. Она принимает текстовый префикс (например, `"test"` или `"user"`).
2. Внутри функции сгенерируйте случайное число в диапазоне от 100 до 999 (используйте `random.randint`).
3. Склейте префикс, случайное число и домен `@qa.ru` через f-строку, чтобы получился адрес вида: `test_554@qa.ru`.
4. Верните этот email через `return`.
5. В основном коде вызовите функцию 3 раза для префикса `"tester"` и напечатайте результаты. Убедитесь, что каждый раз генерируется уникальный адрес!

## **Решение**

import random

def generate_qa_email(prefix):

    rand_num = random.randint(100, 999)
    return f'{prefix}_{rand_num}@qa.ru'

email1 = generate_qa_email('tester')

email2 = generate_qa_email('tester')

email3 = generate_qa_email('tester')

print(email1)

print(email2)

print(email3)


## **Задание 3**
Валидатор длины пароля
Бизнес-требование: По ТЗ безопасности пароль на сайте должен содержать не менее 8 символов. Напишите функцию, которая проверяет это требование.
Что сделать в коде:
1. Напишите функцию `validate_password_security(password)`.
2. Внутри функции измерьте длину строки `password` с помощью встроенной функции `len()`.
3. Если длина строго больше или равна 8, функция должна вернуть строку: `"[PASS] Пароль безопасен."`.
4. Иначе функция должна вернуть строку: `"[FAIL] Пароль слишком короткий! Текущая длина: {длина} знаков."`.
5. В основном коде запросите у пользователя ввод пароля через `input()`. Передайте этот ввод в вашу функцию, сохраните возвращенный результат в переменную и выведите её на экран через `print()`.

## **Решение**

def validate_password_security(password):

    lenpass=len(password)
    if lenpass >= 8:
        return "[PASS] Пароль безопасен"
    else:
        return f"[FAIL] Пароль слишком короткий! Текущая длина: {lenpass} знаков."


pass_input = validate_password_security(input("Введите пароль: "))

print(pass_input)
