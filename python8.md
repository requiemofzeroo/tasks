## **Задание 1**

Матрица пикселей экрана
Бизнес-требование: Автотест проверяет сетку пикселей экрана мобильного приложения (разрешение 3x3), чтобы найти «битые» пиксели.
Что сделать в коде:
1. Напишите внешний цикл по строкам: `for row in range(1, 4):`
2. Внутрь вставьте цикл по колонкам: `for col in range(1, 4):`
3. В самом глубоком блоке выведите координаты проверяемой точки через f-строку: `[SCAN] Проверяем пиксель: Строка {row}, Колонка {col}`.

## **Решение**

        print(f'[SCAN] Проверяем пиксель: Строка {row}, Колонка {col}')

## **Задание 2**

Очистка спам-комментариев на страницах
Бизнес-требование: Автотест имитирует работу модератора. Нам нужно проверить 3 статьи на сайте (статьи №1, №2, №3). Под каждой статьей пользователи оставили по 4 комментария (номера 1, 2, 3, 4). Если номер комментария равен 3 — это спам с рекламой, модератор должен удалить его (пропустить действие), но продолжить проверять остальные комменты под этой статьей.
Что сделать в коде:
1. Напишите внешний цикл перебора статей от 1 до 3. На каждом шаге внешнего цикла печатайте: `--- Модерируем статью №{article} ---`.
2. Напишите внутренний цикл перебора комментариев от 1 до 4.
3. Если номер комментария равен 3, выведите лог: `   [DELETE] Ссылка на казино удалена!` и примените `continue`.
4. Иначе выведите лог: `   [OK] Комментарий №{comment} проверен.`

## **Решение**

for article in range(1,4):

    print(f'--- Модерируем статью №{article} ---')
    for comment in range(1,5):
        if comment == 3:
            print('[DELETE] Ссылка на казино удалена!')
            continue

        print(f'[OK] Комментарий №{comment}')

## **Задание 3**

Сканер уязвимостей портов
Бизнес-требование: Напишите симулятор сканера безопасности (Cybersecurity QA). Мы проверяем 3 сервера (серверы №1, №2, №3). На каждом сервере мы сканируем порты с номерами от 80 до 83. Если сканер натыкается на критическую уязвимость на порту 82, это значит, что вся сеть скомпрометирована — нужно НЕМЕДЛЕННО прекратить сканирование этого сервера И всех последующих серверов (полный останов программы).
Что сделать в коде:
1. Создайте флаг безопасности `network_compromised = False`.
2. Напишите внешний цикл по серверам (от 1 до 3). На входе в него добавьте проверку флага: если он True — прервите внешний цикл через `break`.
3. Внутри напишите цикл по портам: `for port in range(80, 84):`.
4. Внутри напишите проверку: если `port == 82` И при этом `server == 2` (допустим, уязвимость именно на втором сервере), выведите лог `🛑 [ALARM] На сервере {server} обнаружена брешь на порту {port}! Глобальный стоп!` $\rightarrow$ переключите флаг `network_compromised = True` $\rightarrow$ сделайте `break`.
5. Если уязвимости нет, выводите обычный лог: `   [SCAN] Сервер {server}: порт {port} чист.`

## **Решение**

network_compromised = False

for server in range(1, 4):

    if network_compromised:
        break
    for port in range(80, 84):
        if server == 2 and port == 82:
            print(f'🛑 [ALARM] На сервере {server} обнаружена брешь на порту {port}! Глобальный стоп!')
            network_compromised = True
            break
        print(f'[SCAN] Сервер {server}: порт {port} чист.')
