## **Задание 1**
Эмулятор бесконечной прокрутки
Бизнес-требование: Автотест имитирует поведение пользователя, который скроллит ленту новостей на сайте. Нужно пролистать 5 страниц.
Что сделать в коде:
1. Создайте переменную-счетчик `current_page = 1`.
2. Напишите цикл `while`, который работает, пока `current_page` меньше или равен 5.
3. Внутри цикла выводите через f-строку лог: `[INFO] Скроллим страницу №{current_page}`.
4. Обязательно увеличивайте счетчик страниц на 1 внутри цикла (`+= 1`).
5. Вне цикла выведите финальный лог успеха: `[PASS] Лента успешно протестирована.`

## **Решение**

 current_page = 1

 while current_page <= 5:
 
     print(f'[INFO] Скроллим страницу №{current_page}')
     
     current_page += 1

 print('[PASS] Лента успешно протестирована.')

## **Задание 2**
Ожидание загрузки элемента
Бизнес-требование: Автотест проверяет фичу «Smart Wait» (умное ожидание). Наша программа должна опрашивать сервер каждую секунду, ожидая появления элемента на экране, но не более 5 секунд (тайм-аут).
Что сделать в коде:
1. Создайте переменную `is_element_visible = False`.
2. Создайте переменную счетчика секунд: `seconds_passed = 0`.
3. Напишите цикл `while`, который работает, пока `seconds_passed` меньше 5 И элемент НЕ виден (`not is_element_visible`).
4. Внутри цикла:
   • Увеличивайте счетчик секунд на 1.
   • Выводите лог: `Ожидание загрузки... Прошло секунд: {seconds_passed}`.
   • Напишите проверку `if`: если `seconds_passed == 3` (имитируем, что на 3-й секунде элемент наконец-то догрузился бэкендом), переключите флаг `is_element_visible = True`.
5. Вне цикла напишите финальную проверку `if...else`: если элемент в итоге стал видимым, выведите `[PASS] Элемент найден!`, иначе — `[FAIL] Тайм-аут! Элемент не загрузился за 5 секунд.`

## **Решение**

is_element_visible = False
seconds_passed = 0

while seconds_passed < 5 and not is_element_visible:

    seconds_passed += 1
    
    print(f'Ожидание загрузки... Прошло секунд: {seconds_passed}')

    if seconds_passed == 3:
        is_element_visible = True
    if is_element_visible:
        print(f'[PASS] Элемент найден!')
    else:
        print(f'[FAIL] Тайм-аут! Элемент не загрузился за 5 секунд.')


## **Задание 3**

Симулятор скачивания файла
Бизнес-требование: Мы тестируем стабильность скачивания тяжелого отчета из базы данных. Файл весит 100 Мб. Программа должна имитировать скачивание пачками по 25 Мб за раз и логировать прогресс.
Что сделать в коде:
1. Создайте переменную `total_size = 100`.
2. Создайте переменную `downloaded = 0`.
3. Напишите цикл `while`, который работает, пока `downloaded` строго меньше `total_size`.
4. Внутри цикла:
   • Увеличивайте переменную `downloaded` на 25 (имитация скачивания пачки данных).
   • Рассчитайте процент скачивания прямо внутри f-строки или в отдельной переменной: `(downloaded / total_size) * 100`.
   • Выводите лог вида: `Скачано {downloaded} Мб из {total_size} ({процент}%)`.
5. Вне цикла выведите красивый лог успешного завершения теста.


## **Решение**

total_size = 100

downloaded = 0

while downloaded < total_size:

    downloaded += 25
    
    percent = (downloaded / total_size) * 100
    
    print(f'Скачано {downloaded} МБ из {total_size} ({percent}%)')

print(f'[PASS] Отчет скачан. Тест пройден')
