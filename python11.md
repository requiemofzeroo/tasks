## **Задание 1**

Валидатор JSON-ответа профиля
Бизнес-требование: Автотест проверяет профиль пользователя, прилетевший от API в виде словаря. Нам нужно обновить статус и вывести данные.
Что сделать в коде:
1. Создайте словарь: `user_profile = {"username": "qa_racer", "status": "pending", "bugs_found": 12}`
2. Измените значение ключа `"status"` на `"approved"`.
3. Добавьте в словарь новый ключ `"level"` со значением `"Senior"`.
4. Выведите измененный словарь в консоль.

user_profile = {

    "status": "pending",
    "bugs_found": 12
}

## **Задание 2**

Массовая проверка заголовков API в цикле
Бизнес-требование: Для прохождения теста безопасности все заголовки (headers) запроса к серверу не должны быть пустыми. Нам нужно перебрать их в цикле.
Что сделать в коде:
1. Создайте словарь: `headers = {"Content-Type": "application/json", "Authorization": "Bearer_XYZ", "X-Test-Suite": ""}`
2. Напишите цикл `for k, v in headers.items():` для перебора заголовков.
3. Внутри цикла напишите проверку `if`: если значение `v` равно пустой строке `""`, выведите критический лог: `[FAIL] Найден пустой заголовок: {k}! Тест безопасности провален.`

## **Решение**

headers = {

    "Content-Type": "application/json",
    "Authorization": "Bearer_XYZ",
    "X-Test-Suit": ""
}

for keys, values in headers.items():

    if values == "":
        print(f'[FAIL] Найден пустой заголовок: {keys}! Тест безопасности провален')

## **Задание 3**

Детектор мошеннических транзакций
Бизнес-требование: Антифрод-система (тестирование безопасности) зафиксировала ID кошельков, с которых переводились деньги. Если один и тот же кошелек совершил несколько переводов, система считает это подозрительным. Нам нужно проверить, есть ли в списке дубликаты.
Что сделать в коде:
1. Создайте список: `wallet_ids = [1002, 5005, 1002, 7001, 5005, 9003]`
2. Посчитайте длину исходного списка и сохраните в переменную `len_list`.
3. Переведите список во множество `set(wallet_ids)`, посчитайте длину полученного множества и сохраните в переменную `len_set`.
4. Напишите условие `if...else`: если `len_list` НЕ равен `len_set`, выведите лог: `[WARNING] Обнаружены повторные транзакции! Система антифрод заблокировала дубликаты.`. Иначе выведите: `[PASS] Все транзакции чистые.`

## **Решение**

wallet_ids = [1002, 5005, 7001, 5005, 9003]

len_list = len(wallet_ids)

len_set = len(set(wallet_ids))

if len_list != len_set:
    print('[WARNING] Обнаружены повторные транзакции! Система антифрод заблокировала дубликаты.')

else:
    print('[PASS] Все транзакции чистые.')
