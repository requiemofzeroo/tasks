Создайте чек-лист на основе лекции «Особенности мобильного тестирования».

Включите в него ключевые пункты, связанные с:

### **Задание:**
1. Выберите любое мобильное приложение (установленное на вашем
устройстве).
2. Проведите его тестирование, используя составленный вами чек-лист.
3. Отметьте в чек-листе результаты выполнения каждого пункта (например:
«пройдено», «не пройдено», «ошибка», «не применимо» и т.д.).


### **Чек-лист для мобильного приложения:**

**Интерфейс и адаптивность:**

*  Отображение на разных экранах
*  Читаемость текста

**Поворот экрана:**

*  Сохранение состояние при повороте экрана

**Производительность:**

*  Быстрый запуск
*  Плавность работы

**Уведомления:**

*  Переход по нажатию
*  Не мешает при звонке
*  Можно прочитать при заблокированном экране

**Плохое соединение:**

*  Сообщение об ошибке
*  Работа оффлайн

**Фоновый режим:**

*  Не сбрасывает состояние приложения
*  Возобновляется без проблем
