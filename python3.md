Что нужно сделать в коде:
1. Запросите у пользователя ввод пароля через `input()`.
2. Измерьте длину введенного пароля с помощью встроенной функции `len()` (например: `length = len(user_password)`).
3. Напишите условие `if...else`: 
   • Если длина строго больше или равна 6, выведите лог: `[PASS] Пароль валиден.`

user_password = input("Введите пароль: ")

## **Задание №2**

Бизнес-требование: Автотест проверяет фичу «Двухфакторная аутентификация». У пользователя есть всего 3 попытки на ввод правильного СМС-кода. Если попытки исчерпаны, аккаунт временно блокируется.
 
Что нужно сделать в коде:
1. Создайте переменную `correct_code = 7788`.
2. Создайте переменную `attempts_left = 1` (осталась всего одна попытка!).
3. Запросите у пользователя ввод СМС-кода из консоли.
4. Приведите введенный код к типу `int`.
5. Напишите условие `if...else`:
   • Если код совпал с `correct_code`, выведите: `[PASS] Код верный! Вход выполнен.`
   • Если код не совпал, выполните внутри блока `else` два действия:
     а) Снизьте счетчик попыток на 1 (используйте сокращенную запись `-= 1`).
     б) Выведите сообщение через f-строку: `[FAIL] Неверный код! Осталось попыток: {attempts_left}. Аккаунт заблокирован.`

correct_code = 7788

attempts_left = 1

sms_code = int(input("Введите код из СМС: "))

if sms_code == correct_code:

    print("[PASS] Код верный! Вход выполнен.")
else:

    attempts_left -= 1
    
    print(f'[FAIL] Неверный код! Осталось попыток: {attempts_left}. Аккаунт заблокирован')


## **Задание 3**
Бизнес-требование: В автотесте нужно проверить, заполнил ли пользователь обязательные поля в профиле — «Имя» и «Возраст». Если хотя бы одно поле не заполнено (или в возрасте указан 0), система не должна сохранять профиль.
 
Что нужно сделать в коде:
1. Запросите у пользователя ввод имени (`user_name`).
2. Запросите у пользователя ввод возраста в виде строки (`user_age_raw`).
3. Приведите возраст к типу `int` и сохраните в переменную `user_age`.
4. Напишите проверку, используя логический оператор `and` и магию функции `bool()`. Условие должно проверять: строка имени НЕ пустая **И** возраст НЕ равен нулю.
   • Если оба поля заполнены корректно, выведите: `[PASS] Профиль успешно обновлен.`
   • Иначе выведите: `[FAIL] Ошибка: Заполните все обязательные поля корректно!`


user_name = input ("Введите имя: ")

user_age_raw = input("Введите возраст: ")

user_age = int(user_age_raw)

if bool(user_name) and bool(user_age):

    print("[PASS] Профиль успешно обновлен.")
else:

    print("[FAIL] Ошибка: Заполните все обязательные поля корректно!")
