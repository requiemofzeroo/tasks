### **Задание 1:**
**Напишите краткий конспект по тестированию API, ответив на следующие
вопросы:**
1. Что такое API?
2. Что такое POST и GET запросы?
3. Что такое клиент-серверное приложение?
4. Какие программы для тестирования API вы знаете?
5. Что такое REST и SOAP?

# **Ответы на вопросы:**
1. API - это интерфейс, который позволяет программам "общаться" друг с другом
2. GET - запрос на получение данных
   POST - запрос на отправку данных на сервер
3. Клиент-серверное приложение - это приложение (браузер или мобильное приложение) где клиент отправляет запросы на сервер, а сервер обрабатывает их и возвращает ответ.
4. Postman, SoapUI
5. REST - стиль API, который работает через HTTP и JSON
   SOAP - протокол, который использует XML
