## **Задание #3**


Бизнес-требование: Система одобряет транзакцию, если сумма перевода не превышает баланс, ЛИБО если у пользователя включен лимит овердрафта (кредитный лимит), но при условии, что аккаунт не заблокирован службой безопасности.
• Входные данные (запросить через input и привести к нужным типам):
  1. Сумма перевода (`amount` -> int)
  2. Баланс счета (`balance` -> int)
  3. Есть ли овердрафт (`has_overdraft` -> 1 или 0 -> bool)
  4. Заблокирован ли аккаунт (`is_blocked` -> 1 или 0 -> bool)
• Напишите условие: Транзакция успешна, если аккаунт НЕ заблокирован (`not is_blocked`) И при этом выполняется одно из двух условий: сумма меньше или равна балансу ИЛИ есть овердрафт. (Используйте правильную расстановку скобок!).
• Если все ок, выведите: "Транзакция одобрена". Иначе: "Транзакция отклонена".


amount = int(input("Введите сумму перевода: "))

balance = int(input("Введите баланс счета: "))

has_overdraft = bool(input("Есть ли овердрафт? (1 - да, 0 - нет) - "))

is_blocked = bool(input("Аккаунт заблокирован? (1 - да, 0 - нет) - "))

if not is_blocked and (amount <= balance or has_overdraft):

    print("Транзакция одобрена.")
else:

    print("Транзакция отклонена.")
