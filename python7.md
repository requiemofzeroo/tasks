## **Задание 1**

Ограничитель загрузки страниц
Бизнес-требование: Автотест проверяет пагинацию каталога (всего на сайте 10 страниц), но по ТЗ нам нужно проверить только первые 4 страницы.
Что сделать в коде:
1. Напишите цикл `for page in range(1, 11):` (перебор страниц от 1 до 10).
2. Внутри цикла напишите проверку `if`: если `page` стал равен 5, выведите сообщение `[INFO] Лимит первых страниц достигнут. Останавливаем тест.` и прервите цикл с помощью `break`.
3. Ниже этой проверки (но внутри цикла) выводите лог: `[PASS] Страница каталога {page} успешно проверена.`

## **Решение**

 for page in range(1, 11):
 
     if page == 5:
         print(f'[INFO] Лимит первых страниц достигнут. Останавливаем тест.')
         break
     print(f'[PASS] Страница каталога {page} успешно проверена')


## **Задание 2**

Посимвольный валидатор e-mail
Бизнес-требование: При вводе email в форму на сайте автотест должен проверить, что пользователь обязательно вставил символ «собачки» (`@`). Если символа нет — форма невалидна.
Что сделать в коде:
1. Запросите у пользователя ввод email через `input()`.
2. Создайте флаг `has_at_symbol = False`.
3. С помощью цикла `for` переберите введенный email посимвольно.
4. Если нашли символ `@`, переключите флаг `has_at_symbol = True` и досрочно завершите цикл через `break` (зачем искать дальше, если символ уже найден?).
5. Вне цикла проверьте флаг через `if...else`: если он остался False, выведите `[FAIL] Ошибка! В email отсутствует символ @`. Иначе выведите `[PASS] Email прошел базовую проверку.`

## **Решение**

user_email = input("Введите свою почту: ")

has_at_symbol = False

for char in user_email:

    if char == "@":
        has_at_symbol = True
        break

if not has_at_symbol:

    print(f'[FAIL] Ошибка! В e-mail отсутствует "@"')
else:
    print(f'[PASS] E-mail прошел базовую проверку')


## **Задание 3**

Детектор запрещенных IP-адресов в логах
Бизнес-требование: Скрипт анализирует текстовую строку с номерами серверов. Номер сервера `3` заблокирован Роскомнадзором, запросы к нему слать нельзя — тест должен пропустить его, но продолжить проверку остальных серверов. Если натыкаемся на несуществующий сервер `9`, это признак взлома — нужно немедленно прекратить работу скрипта.
Что сделать в коде:
1. Создайте строку серверов: `servers_queue = "12345978"` (шагаем по ней посимвольно).
2. Напишите цикл `for server in servers_queue:`. Внутри переведите символ в число: `server_id = int(server)`.
3. Напишите проверку: если `server_id == 3`, выведите лог `[WARN] Сервер 3 заблокирован! Пропускаем.` и используйте `continue`.
4. Напишите вторую проверку: если `server_id == 9`, выведите лог `[CRITICAL] Обнаружен вредоносный сервер 9! Мгновенная остановка!` и используйте `break`.
5. Ниже проверок внутри цикла выводите обычный лог: `[SUCCESS] Запрос к серверу {server_id} успешно отправлен.`

## **Решение**

servers_queue = '12345678'

for server in servers_queue:

    server_id = int(server)
    if server_id == 3:
        print('[WARN] Сервер 3 заблокирован! Пропускаем')
        continue
    if server_id == 9:
        print(f'[CRITICAL] Обнаружен вредоносный сервер 9! Мгновенная остановка!')
        break
    print(f'[SUCCESS] Запрос к серверу {server_id} успешно отправлен.')
