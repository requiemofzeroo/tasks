## **Задание 1**
Конструктор логов для Jenkins
Бизнес-требование: Напишите функцию-помощник, которая красиво оформляет логи тестов в CI/CD.
Что сделать в коде:
1. Создайте функцию `format_log(message, level="INFO")`.
2. Функция должна собирать и возвращать через `return` строку формата: `[{level}] {message}`.
3. В основном коде вызовите функцию дважды: первый раз передайте только текст `"Тест запущен"` (уровень должен стать дефолтным INFO). Вторым вызовом передайте текст `"Критический баг"` и явно измените уровень на `"ERROR"` через ключевой аргумент. Результаты выведите на экран.

## **Решение**

def format_log(message, level='INFO'):

    return f'[{level}] {message}'

print(format_log(message="Тест запущен"))

print(format_log(message="Критический баг", level='ERROR'))


## **Задание 2**
Валидатор API-запроса с таймаутом
Бизнес-требование: Переиспользуемый хелпер должен имитировать отправку запроса на сервер с настраиваемым временем ожидания.
Что сделать в коде:
1. Создайте функцию `send_api_request(endpoint, timeout=3, secure_connection=True)`.
2. Функция должна собирать через f-строку и возвращать лог вида: `[API] Запрос к {endpoint} отправлен. Тайм-аут: {timeout} сек. Безопасность: {secure_connection}`.
3. В основном коде выполните три разных вызова:
   • Вызов 1: Передайте только эндпоинт `"/login"` (остальное дефолт).
   • Вызов 2: Передайте эндпоинт `"/metrics"` и измените таймаут на `10` секунд.
   • Вызов 3: Измените параметр `secure_connection` на `False` с помощью ключевого вызова, передав эндпоинт `"/debug"`.
4. Напечатайте все три результата.

## **Решение**

def send_api_request(endpoint, timeout=3, secure_conntection=True):

    return f'[API] Запрос к {endpoint} отправлен. Тайм-аут: {timeout} сек. Безопаснсоть: {secure_conntection}'


print(send_api_request(endpoint="/login"))

print(send_api_request(endpoint="/metrics", timeout=10))

print(send_api_request(endpoint="/debug", secure_conntection=False))

## **Задание 4**
Напишите функцию, которая определяет високосный год или нет

## **Решение**

def is_leap_year(year):

    return year % 4 == 0 and (year % 100 != 0 or year % 400 == 0)

print(is_leap_year(1900))

print(is_leap_year(2100))

print(is_leap_year(2000))

print(is_leap_year(2024))
