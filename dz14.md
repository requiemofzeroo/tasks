# **Отчет по тестированию Usability приложения SoundCloud (Web - версия)**

18.01.2026 года была протестирована Usability Web - версия приложения SoundCloud на следуюшие качественные составляющие:

-  **Обучаемость:**

  Сразу после входа на сайт понятно, где находится поиск и кнопка воспроизведения. Основные функции хорошо заметны и интуитивно понятны. Пользователь может начать слушать музыку без регистрации. Однако некоторые дополнительные возможности, такие как загрузка треков и работа с профилем автора, могут быть не сразу очевидны.

-  **Эффективность:**

После ознакомления с интерфейсом пользователь может быстро выполнять основные действия. Поиск треков и исполнителей работает удобно и точно. Управление воспроизведением происходит без лишних действий. Основные элементы интерфейса всегда находятся под рукой. Иногда большое количество рекомендаций может немного отвлекать от основной задачи.

-  **Запоминаемость:**

Интерфейс SoundCloud хорошо запоминается. Расположение элементов остаётся логичным и привычным. Даже после перерыва пользователь быстро вспоминает, как пользоваться сайтом. Навигация не вызывает затруднений. Повторное использование сервиса не требует повторного обучения.

-  **Ошибки:**

При работе с сайтом SoundCloud у пользователей иногда возникают ошибки. Некоторые жалуются, что сайт может работать нестабильно после обновлений. Бывают проблемы с входом в аккаунт, когда кнопки не нажимаются или страница загружается неправильно. Иногда поиск или плейлисты работают медленно и требуют обновления страницы. Также встречаются сложности с получением кода подтверждения при входе. Эти ошибки не критичны, но могут мешать комфортному использованию сайта.

-  **Удовлетворенность:**

Для некоторых пользователей отсутствует интерфейс на родном языке, что может вызывать трудности при использовании сервиса. Из-за этого общий уровень удобства и комфорта для части аудитории может снижаться.
