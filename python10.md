## **Задание 1**

Чистильщик телефонных номеров
Бизнес-требование: Автотест UI-формы получает номер телефона в виде строки с дефисами: `"8-999-111-22-33"`. Бэкенд принимает только чистые цифры.
Что сделать в коде:
1. Создайте переменную со строкой номера.
2. Используйте метод `.replace()`, чтобы заменить все дефисы на пустые строки `""`.
3. Выведите итоговую чистую строку в консоль.

## **Решение**

phone_number = '8-999-111-22-33'

new_phone_number = phone_number.replace('-', '')

print(new_phone_number)

## **Задание 2**

Вырезка токена из URL
Бизнес-требование: После авторизации автотест получает URL-адрес страницы, в хвосте которого зашит секретный токен: `https://qa.ru?secret=my_token`. Нам нужно достать этот токен для следующих тестов.
Что сделать в коде:
1. Создайте переменную с указанным URL.
2. С помощью метода `.find()` найдите индекс начала токена (индекс буквы `s` в слове `my...`, которая идет сразу после знака `=`).
3. Используя срез строки от найденного индекса до конца, вырежьте токен в отдельную переменную и напечатайте её.

## **Решение**

url_adress = "https://qa.ru?secret=my_token"

index_url = url_adress.find("my")

token_url = url_adress[index_url:]

print(token_url)


## **Задание 3**

Фильтр тестовых логов
Бизнес-требование: Из лог-файла автотест вытащил строку со списком выполненных шагов: `"Шаг1;Шаг2;Ошибка:404;Шаг3"`. Нам нужно распарсить эту строку и проверить её безопасность.
Что сделать в коде:
1. Создайте переменную с указанной строкой лога.
2. Примените метод `.split(";")`, чтобы разрезать строку по точке с запятой. Результат сохранится в список.
3. Напишите условие `if...else` с использованием оператора принадлежности `in`: проверочное слово `"Ошибка:404"` есть в вашем новом списке?
   • Если есть, выведите: `[FAIL] Тест упал. Обнаружен критический баг.`
   • Если нет, выведите: `[PASS] Все шаги выполнены успешно.

## **Решение**

log_list = 'Шаг1;Шаг2;Ошибка:404;Шаг3'

log_split = log_list.split(';')

print(log_split)

if 'Ошибка:404' in log_split:

    print('[FAIL] Тест упал. Обнаружен критический баг.')
else:

    print('[PASS] Все шаги выполнены успешно.')
