## **Контрольные вопросы**
1. Ищет файлы и папки в системе.
2. Основные опции:
-  -name - поиск по имени
-  -iname - тоже поиск по имени, только уже без учета регистра
-  -type f/d - ищет только (f) - файлы, (d) - каталоги
-  size - размер файла
3.  find . -name "*.log" -exec rm {} \;
4.  Превращает список в агрументы для другой команды
5.  Ищет строки, содержащие заданный текст внутри файлов
6.  Основные опции:
-  -r - рекурсивно по папкам
-  -i - игнорировать регистр
-  -v - показать строки, не содержащие шаблон
-  -n - показать номер строки
7.  ls -l | grep ".txt"
