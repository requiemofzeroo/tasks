# Bug-report
### 1.**ID:** 01
### 2. **Заголовок:** Цена пиццы со скидкой отображаются без скидки в корзине
### 3. **Описание:** При добавлении пиццы со скидкой в корзину - итоговая цена в корзина без учета скидки:
- 3.1 **Предусловие:** На странице каталога пиццы отображается карточка пиццы со скидкой
- 3.2 **Тестовые данные:** Скидка на пиццу дня - "Пепперони"
- 3.3 **Шаги воспроизведения:**
- 3.3.1 Зайти на сайт - https://slqamsk.github.io/cases/pizza/v09/index.html?cart=2%3A2
- 3.3.2 Добавить пиццу дня - "Пепперони" в корзину
- 3.3.3 Перейти в корзину
- 3.4 **Ожидаемый результат:** Правильно рассчитанная скидка на пиццу дня - 495 рублей
- 3.5 **Наблюдаемый результат:** В корзине отображается полнная ценна пиццы без скидки
### 4. **Найден в версиях:** v09
### 5. **Критичность:** Ошибка ценнобразования влияет на кол-во посетителей сайта
### 6. **Приоритет:** Высокий (критично для продаж)
