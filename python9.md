## **Задание 1**

Поиск битого пикселя через for...else
Бизнес-требование: Автотест сканирует 4 экрана интерфейса приложения (номера от 1 до 4) на наличие критического бага отображения. Если на экране №3 обнаружен сбой, тест прерывается. Если все экраны проверены — выводится отчет об успешном UI-тестировании.
Что сделать в коде:
1. Напишите цикл `for screen in range(1, 5):`.
2. Внутри запрашивайте у пользователя статус экрана: `is_broken = int(input(f"Экран {screen} сломан? (1 - да, 0 - нет): "))`. Приведите к bool.
3. Если `is_broken` равен `True` И при этом `screen == 3`, выведите сообщение `🛑 [FAIL] Тест упал на экране 3!` и сделайте `break`.
4. Напишите блок `else` для этого цикла. Внутри него выведите: `💚 [PASS] Все экраны отображаются корректно. Тест пройден!`

## **Решение**

for screen in range(1,5):

    is_broken = bool(int(input(f'Экран {screen} сломан? (1 - да, 0 - нет): ')))
    if is_broken and screen == 3:
        print('🛑 [FAIL] Тест упал на экране 3!')
        break
else:

    print('[PASS] Все экраны отображаются корректно. Тест пройден!' )


## **Задание 2**

Симулятор подбора ключей API через вложенный `while`
Бизнес-требование: Тестировщик проверяет защиту API от брутфорса. Нужно сымитировать подбор ключей доступа. У нас есть 2 секретных сервера (`server = 1` и `server = 2`). К каждому серверу скрипт пытается подобрать ключ, делая максимум по 2 попытки (`attempt = 1` и `attempt = 2`).
Что сделать в коде:
1. Напишите вложенный цикл `while`. Внешний цикл по `server` работает, пока `server <= 2`.
2. Внутри внешнего цикла (перед внутренним) обязательно сбрасывайте `attempt = 1`.
3. Внутренний цикл работает, пока `attempt <= 2`.
4. На каждом шаге запрашивайте у пользователя ввод секретного ключа (строка): `key = input(f"Сервер {server}, Попытка {attempt}. Введите ключ: ")`.
5. Если пользователь ввел правильный ключ `"OPEN123"`, выведите `[ACCESS] Доступ к серверу {server} получен!` и сделайте `break` (чтобы остановить попытки для текущего сервера и перейти к следующему).
6. Не забывайте увеличивать счетчики: `attempt += 1` во внутреннем цикле и `server += 1` во внешнем!

## **Решение**

server = 1

while server <=2:

    attempt = 1
    while attempt <=2:
        key = input(f'Сервер {server}, Попытка {attempt}. Введите ключ: ')
        if key == 'OPEN123':
            print(f'[ACCESS] Доступ к серверу {server} получен!')
            break
        attempt += 1
    server +=1
