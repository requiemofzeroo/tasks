* `__environment` — строка. Стартовое значение: `"STAGE"`.

* Если среды нет в списке — проигнорировать изменение и вывести предупреждение: `"[VALIDATION ERROR] Попытка переключиться на несуществующий стенд!"`.
 
5️⃣ Реализуйте метод-сеттер `set_threads(count)`.

* Если проверка пройдена — обновить `__threads_count`.
* Если проверка провалена (передан не тот тип или число вне диапазона) — оставить старое значение и вывести лог: `"[VALIDATION ERROR] Недопустимое количество потоков!"`.

## **Решение**

class TestRunnerConfig:
    def __init__(self):
    
        self.__environment = "STAGE"
        self.__allowed_envs = ["STAGE", "PRERPOD", "PROD"]
        self.__threads_count = 1

    @property

    def runner_status(self):
        if self.__threads_count == 1:
            return f"Тесты запущены в однопоточном режиме на стенде {self.__environment}"
        if self.__threads_count > 1:
            return f"Параллельный запуск: {self.__threads_count} потоков на стенде {self.__environment}"


    def change_evniroment(self, new_env):
        if new_env in self.__allowed_envs:
            self.__environment = new_env
            print("[CONFIG] Стенд успешно переключен")

        else:
            print("[VALIDATION ERROR] Попытка переключиться на несуществующий стенд!")

    def set_threads(self, count):
        if type(count) is int and 1 <= count <= 8:
            self.__threads_count = count
        else:
            print("[VALIDATION ERROR] Недопустимое количество потоков!")

testrunner.set_threads("пять")

testrunner.set_threads(99)

testrunner.change_evniroment("PREPROD")

testrunner.set_threads(4)

print(testrunner.runner_status)
